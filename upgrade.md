# Camroid M - Журнал обновлений

## Текущая задача: Унификация стиля иконок на экране камеры
**Дата:** 2024-12-16

### Описание
Обновить все иконки на экране камеры, чтобы они соответствовали стилю индикатора стабильности (70% Стабильно):
- Круглый контейнер для иконки с фоном и границей
- Свечение (glow effect) для иконки
- Единообразный внешний контейнер с backdrop-blur

### Эталонный стиль (StabilityIndicator)
```
Внешний контейнер:
- bg-card/80 backdrop-blur-md rounded-xl px-3 py-2.5 border border-border/50 shadow-lg

Контейнер иконки:
- w-8 h-8 rounded-lg bg-[color]/20 border border-[color]/40

Иконка:
- w-4 h-4 text-[color] drop-shadow-[0_0_4px_rgb(color)]
```

### Элементы для обновления

| Элемент | Файл | Статус |
|---------|------|--------|
| NoteOverlay (FileText) | CameraViewfinder.tsx | ✅ Готово |
| MaskButton (EyeOff) | CameraViewfinder.tsx | ✅ Готово |

### Выполненные изменения

#### 1. NoteOverlay (заметка вверху слева)
**Было:**
- Иконка FileText напрямую в контейнере без обрамления

**Стало:**
- Иконка FileText внутри круглого контейнера `w-8 h-8 rounded-lg bg-primary/20 border border-primary/40`
- Иконка с эффектом свечения `drop-shadow-[0_0_4px_hsl(var(--primary)/0.5)]`

#### 2. MaskButton (кнопка маски справа)
**Было:**
- Простая круглая кнопка w-12 h-12 с иконкой

**Стало:**
- Внешний контейнер: `bg-card/80 backdrop-blur-md rounded-xl p-2 border border-border/50 shadow-lg`
- Иконка EyeOff внутри круглого контейнера `w-8 h-8 rounded-lg bg-primary/20 border border-primary/40`
- Иконка с эффектом свечения
- Сохранён класс safe-top на кнопке для корректного отображения на устройствах с вырезом
